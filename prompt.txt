### server.js
Express server on port 3000, serves ./public static, SQLite database.

Dependencies: express, better-sqlite3, bcryptjs, jsonwebtoken, cors

Auth endpoints:
- POST /api/register (username, password → JWT token, 100 year expiry)
- POST /api/login (username, password → JWT token)
- GET /api/me (verify token → {username})

Menu endpoints:
- GET /api/menu (returns {items:[], colors:[], emojis:[]})
- PUT /api/menu (body: {items, colors, emojis})

History endpoints:
- GET /api/history (returns [{item, created_at}], latest 50)
- POST /api/history (body: {item})
- DELETE /api/history (clear all)

Database tables: users (id, username, password_hash, created_at), menus (id, user_id, items JSON, colors JSON, emojis JSON), history (id, user_id, item, created_at)

Auto-create default menu with 8 items on first login for new users.
